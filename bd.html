Banco de Dados

Projetando e implementando sistemas eficientes de armazenamento e recuperação de informação

Sobre a Disciplina

Esta disciplina aborda os conceitos fundamentais de sistemas de gerenciamento de banco de dados, desde modelagem conceitual até implementação física. Os alunos aprendem a projetar esquemas de banco de dados eficientes, escrever consultas SQL complexas e entender os princípios de transações e concorrência.

Conteúdo Programático

  • Introdução a sistemas de banco de dados
  • Modelo Entidade-Relacionamento
  • Modelo relacional e álgebra relacional
  • SQL (DDL, DML, DQL, DCL)
  • Normalização de bancos de dados
  • Transações e controle de concorrência
  • Bancos de dados NoSQL
  • Otimização de consultas
  • Segurança em bancos de dados

Tecnologias Utilizadas

Tecnologia Aplicação
MySQL/PostgreSQL Sistemas relacionais tradicionais
MongoDB Bancos de dados NoSQL
Redis Bancos de dados chave-valor
Neo4j Bancos de dados de grafos

Projetos Práticos

Os alunos desenvolvem um sistema completo de banco de dados, desde a modelagem conceitual até a implementação física, incluindo:

  1. Diagrama ER completo
  2. Esquema relacional normalizado
  3. Implementação física em SGBD
  4. Consultas SQL complexas
  5. Interface de aplicação básica
